LocalStorage 究竟是做什麼的？

自從 HTML5 使用後，網頁就越來越多功能性可以使用。平常我們使用購物平台會將商品放入購物車中，當我離開網頁再回來時你有發現商品還在嗎？（有些平台就算沒登入帳號仍舊可以看到你購物車中的商品）這就是 LocalStorage 的功能，簡單來說就是「暫存在本地的資料」，通常大概約 5MB 的容量，但也依據瀏覽器不同提供本地的儲存容量也不同。

# Cookie / LocalStorage / SessionStorage

# Cookie

在還沒有 localStorage 的時候是存在 cookie 之中，但是 cookie 只提供 4K 的大小，且有時效性的問題。

  • 儲存容量 4KB
  • 與 Server 溝通時需每次帶 Request，儲存過多的資料會影響效能
  • 應用上有帳號登入、購物車、身份驗證

# LocalStorage

取代 Cookie 多數功能，除非特別移除不然就會一直存在瀏覽器上，沒有時效性問題（可永久保存）。

  • 儲存容量 5MB
  • 不用與 Server 溝通，資料僅儲存在瀏覽器中
  • 應用上有購物車、分頁、收藏等等網頁動作

# SessionStorage

功能與 localStorage 大同小異，最大的差別是 SessionStorage (opens new window) 當網頁關掉時，所有數據就會被清空。

# LocalStorage 使用

最為重要的是 LocalStorage 儲存方式是以 Object 的方式 Key/Value 且只接受 String。

# setItem

localStorage.setItem(key, value)

這邊使用 MDN 上的簡單範例

存入 storage 中

localStorage.setItem('myCat', 'Tom');//存入資料(key, value)

# getItem

抓取 key 值回傳 value

var cat = localStorage.getItem('myCat');
//此時 cat 值為 'Tom'

# removeItem

移除暫存

localStorage.removeItem('myCat');

# clear

清除所有物件

localStorage.clear();

而這些動作都可以在本機上查看操作，以下為範例

# 範例 - 儲存你的生日

codepen (opens new window)

<div class=container>
    <input type="text" class="inputText" placeholder="請輸入生日"><br>
    <div class="gorup-btn">
      <button type="button" class="btn btn-secondary saveName">存入資料</button>
      <button type="button" class="btn btn-secondary getName">取出資料</button>
    </div>
    <h3 class="h2"></h3>
  </div>

# 拆解步驟

  1. 先抓取各個節點
let saveBtn = document.querySelector('.saveName')
let getBtn = document.querySelector('.getName')
let input = document.querySelector('.inputText')
let h2 = document.querySelector('.h2')
  1. 利用監聽事件將輸入的值存入 localstorage 中，以防有未輸入可能性記得多加一道驗證
saveBtn.addEventListener('click', () => {
  let inputValue = input.value // 
  if(inputValue.trim().length === 0) {
    return alert('請輸入生日！')
  } else {
    localStorage.setItem('Birthday', inputValue)
    h2.innerHTML = '已儲存！'
  }
})
  1. 一樣是監聽事件當使用者按下取出資料時，getItem 會抓取 key 值回傳 value（注意兩者都是字串）
getBtn.addEventListener('click', ()=>{
  let getInput = localStorage.getItem('Birthday')
  alert('你的生日是  ' + getInput)
})

可以利用 DevTool 來查看我們是否有存取進去以及清除
