# 目標

RGB 轉換成 16 進制後利用滑動的方式做一個與背景連動的色碼轉換器

# 想法

我會有三個 slider 各代表 R、G、B 三原色，當我滑動的話會依據三個不同滑動的值（依據色碼規則最大值只能 255 最小值 0）變更背景顏色此時我的數字也會跟著變動。

# 步驟拆解

  1. 先將 html 版面排好自己要的樣式以及賦予 class、id
  2. 抓到三個 silder 節點的值
  3. 轉換成 16 進制的規格
  4. 組成色碼（#fffff）
  5. 將背景、文字跟著我的 slider 變動

# HTML - input range 

<input type="range" value="255" min="0" max="255" id=red onchange="changeRange()">
<input type="range" value="255" min="0" max="255" id=green onchange="changeRange()">
<input type="range" value="255" min="0" max="255" id=blue onchange="changeRange()">

平常我們使用 input 都是讓使用者輸入，在 html 上我們通常會這樣寫<input type="text">，而今天是要做成滑動取值的方式，此時的 type 會變成範圍的 input="range" (opens new window)，再設定最大值只能到 255 以及最小值 0，因為是在 slide 上做動作，所以需要加上各自的 ID 好抓取節點。

# onchange

這邊可以看到一個事件叫做onchange (opens new window)，撇除掉像這種下拉選單或是複選等等在 input 上的動作常用以外，不禁好奇 onchange 和 onclick 上的差別，查了一下在 Quora 上有人討論了這個問題 (opens new window)，可以看到這邊寫說

the trigger for onclick is the click on the element and the trigger for onchange is the changing of the value of the element.

這也為什麼我們使用監聽事件時，會常使用 event.target 的原因，我們是要對節點做事件處理，而不是對節點的值。

# 十六進制轉換成色碼

# 抓取 DOM

回到步驟拆解第二步，我們先抓取三個 slide 的節點

let red = document.querySelector('#red')
let green = document.querySelector('#green')
let blue = document.querySelector('#blue')

# 抓取 RGB 各值

當我們對 slide 做動作的時候，會觸發 onchange 的函式。當我們滑動時都會有滑動到一個點的值，利用 value 將值存取到變數，這邊要確保他抓取的數值是整數需要加上 parseInt() (opens new window)

function changeRange() {
    //將字串轉整數字串
    let redValue = parseInt(red.value)
    let greenValue = parseInt(green.value)
    let blueValue = parseInt(blue.value)
}

# 轉成十六進制

當我們抓到 RGB 各 Value 時，要將值做十六進制轉換的動作，另外寫一個函式叫做 hex （參數　number 之後會傳入值），toString() (opens new window) 在括號內放入 16 會回傳轉換後的字母或數字，而如果小於十六進制的話我們就自動幫它補上一個 0 ，好來符合色碼規則。

function hex(number) {
    // 轉換成十六進制
    let hex = number.toString(16);
    // console.log(hex)
    if (number < 16) {
        hex = "0" + hex;
    }
    return hex;
}

# 組成色碼格式更換背景顏色

回到第一個函式將三個的值組成我們常用的色碼存到變數中，當我們一移動 slide 背景就馬上替換顏色，這邊用到 backgroundColor (opens new window) 動態更改背景樣式。

function changeRange() {
    //將字串轉整數字串
    let redValue = parseInt(red.value)
    let greenValue = parseInt(green.value)
    let blueValue = parseInt(blue.value)
    //組合三個 Value
    let color = "#" + hex(redValue) + hex(greenValue) + hex(blueValue);
    //插入背景上
    document.body.style.backgroundColor = color;
}

最後將 RGB 數值跟著一開始的文字做變動，這樣就大功告成了

function changeRange() {
    //將字串轉整數字串
    let redValue = parseInt(red.value)
    let greenValue = parseInt(green.value)
    let blueValue = parseInt(blue.value)
    //組合三個 Value
    let color = "#" + hex(redValue) + hex(greenValue) + hex(blueValue);
    //插入背景上
    document.body.style.backgroundColor = color;
    //文字做變動
    document.querySelector("#hex-label").innerText = color;
    document.querySelector(".red-label").innerText = redValue;
    document.querySelector(".green-label").innerText = greenValue;
    document.querySelector(".blue-label").innerText = blueValue;
}
