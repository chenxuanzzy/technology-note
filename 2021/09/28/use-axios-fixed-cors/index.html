開發上如果在本地端 call api 的話會有可能遇到跨域（CORS） (opens new window)的問題，通常這種時候 Error 都會說要在 header 上加 Access-Control-Allow-Origin response header，但是如果後端沒辦法干涉的話前端該怎麼做？

其實有很多方法！可以參考以下文章

使用跨域代理伺服器(CORS PROXY)，解決讀取第三方網站資料問題﹍實作範例 (opens new window)

這邊我介紹文中其中一個叫做 cors-anywhere (opens new window)

# 範例

作法很簡單，就是將跨域的免費網址加在你的 API 前面就可以了

CodePen 範例 (opens new window)

這邊我串接的是政府開放的 api 且會遇到跨域問題

https://industry.tms.gov.tw/opendata/tmsind.json

我用 Vuetify Table 作為畫面的呈現

<div id="app">
  <v-app id="inspire">
    <v-data-table
      :headers="headers"
      class="elevation-1"
      :items="TableData"
    ></v-data-table>
  </v-app>
</div>

new Vue({
  el: "#app",
  vuetify: new Vuetify(),
  data() {
    return {
      orgData: {}, // 原始資料
      dealData: [], //處理後的資料
      TableData: [], //放入表格內的資料
      headers: [
        { text: "Address1", value: "address" },
        { text: "BusinessHours", value: "business" },
        { text: "Charges", value: "charges" },
        { text: "CompanyName1", value: "name1" },
        { text: "CompanyName2", value: "name2" },
        { text: "CompanyNo", value: "companyNo" },
        { text: "Email", value: "email" },
        { text: "LatLng", value: "latlng" },
        { text: "SportType", value: "sportType" },
        { text: "Tel", value: "tel" },
      ],
    };
  },
  methods: {
    getApi() {
      const cors = "https://cors-anywhere.herokuapp.com/";
      const url = "https://industry.tms.gov.tw/opendata/tmsind.json";
      axios.get(`${cors}${url}`).then((res) => {
        this.orgData = JSON.parse(JSON.stringify(res.data));
        Object.values(this.orgData).forEach((value) => {
          this.dealData.push(value);
        });
        // console.log('org',this.dealData)
        let newObj;
        for (let i = 0; i < this.dealData.length; i++) {
          if (!newObj) {
            newObj = {
              address: this.dealData[i].Address1,
              business: this.dealData[i].BusinessHours,
              charges: this.dealData[i].Charges,
              name1: this.dealData[i].CompanyName1,
              name2: this.dealData[i].CompanyName2,
              companyNo: this.dealData[i].companyNo,
              email: this.dealData[i].Email,
              latlng: this.dealData[i].LatLng,
              sportType: this.dealData[i].SportType,
              // sportTypeOne:'10',
              tel: this.dealData[i].Tel,
            };
          }
          this.TableData.push(newObj);
        }
        console.log("se", this.TableData);
      });
    },
  },
  created() {
    this.getApi();
    // this.dealWithChgData()
  },
});
