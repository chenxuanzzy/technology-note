Question： 因為 php $_REQUEST 回傳的資料格式不是 JSON ，所以需要再另外在 header 帶上 Content-Type

const service = axios.create({
  //$_REQUEST 回傳格式非 JSON 需特別帶 Content-Type
  headers: {
    "Content-Type": "application/x-www-form-urlencoded;charset=utf-8",
  },
  timeout: 50000, // request timeout
});

# 2021/09/03（五）

  • git

    • 高見龍-為你自己學 git (opens new window)

    • 終端機

      • 查看資料夾內所有檔案以及權限ls -al

        ls 指令可列出在目前目錄所有的檔案及目錄，後面接的 -al 參數，a 是指連小數點開頭的檔案（例如.gitignore）也會顯示，l 則是完整檔案的權限、擁有者以及建立、修改時間

      • 也可以自己設定 Mac 下使用 ll 指令 (opens new window)

      • cp fileName.html(要複製的檔案) newName.html(複製出來的新的檔案)

      • mv fileName.html newName.html(更改出來的新的檔案名稱)

      • control + L 清除畫面
