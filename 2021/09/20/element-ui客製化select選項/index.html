有在使用前端框架的應該都知道這套 Element-ui (opens new window) 的套件，是基於 Vue 的框架所製作的，除了友善的中文文件以外，裡面有很多現成網頁功能可以直接套用，算是與 Vuetify 一樣最多人所使用的開發好工具之一。

這邊紀錄一下在工作上所遇到的客製化問題，工作上很多都是使用套件再客製化，每次的客製化都不同，大家都需求都很不一樣呢（汗顏）。

# Select 新增全選選項

官方這邊的 Select (opens new window) 下拉雖然有特定選項 disabled 的功能可以取用，但是卻沒有全選與全不選的功能，所以這部份我們就自己動手增加吧！

詳細可參閱 Codepen (opens new window)

這邊是直接取用官網選擇器去修改，Html 的部分就不多做說明了，概念上不會太難，其實就是去控制選項上的 disabled 動作而已。

# Step1. 新增 disabled 選項

將 select 綁定 cityName，這樣當我們選取的時候 cityName 會增新被選取的物件

<div id="app">
  <template>
    <el-select v-model="cityName" multiple placeholder="請選擇">
      <el-option
        v-for="city in options"
        :key="city.value"
        :label="city.label"
        :value="city.value"
        :disabled="city.disabled"
      >
      </el-option>
    </el-select>
  </template>
</div>

# Step3. 使用 watch 監聽變動

利用 watch 來監聽 cityName 這個陣列 watch 官方文件 (opens new window)

這邊 val 和 oldVal 是 watch 本身帶有的屬性，val 會印出你現在選取到的陣列， oldVal 則是你選取之前的陣列，如果是第一筆選取，oldVal 自然為空陣列。

以下程式上的大概解釋是說當我包含有 value:0 的選項 == true 時，去控制 disabled 的開與關。

watch: {
    cityName(val, oldVal) {
      let currentStatus = val.includes("0");
      let oldStatus = oldVal.includes("0");
      if (currentStatus == true && oldStatus == false) {
        // 現在選取到的如果包含 0 的話
        this.options.forEach((item, index) => {
          if (index > 0) {
            item.disabled = true;
            //除了全選選項都顯示關的狀態
          }
        });
        this.cityName = [];
        this.cityName.push("0");
        //將 cityName 全清空只放入 0
        //如果不多做清空的動作的話
        //當你選取其他縣市選項之時，再去選全選會無法取消其他選到的縣市
      } else if (currentStatus == false && oldStatus == true) {
        //當我再次點選全選時，所有開關打開
        this.options.forEach((item, index) => {
          if (index > 0) {
            item.disabled = false;
          }
        });
      }
    }
  }
