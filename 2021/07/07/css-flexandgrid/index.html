CSS 筆記 - Flex 、 Grid 佈局

Chen Xuan
2021-07-07
  • CSS

Flexbox 和 Grid 佈局可以說是當代排版很夯的方法，這邊紀錄一下，內容也是 Alpha Camp 的課程加上自己的一些實作上遇到的問題（Flex 我花了一整天才理解且實作練習，把過程紀錄一下）。

# Flexbox 和 Grid 的差異

我們先定義一下兩者的差異

  • Flexbox 用於一維排版，適用於容器內的內容跟內容之間的間距或對齊方式。

  • Grid (opens new window) 用於二維以上的排版，用於整個版面區塊式的排版較佳。

# Flex

# Flex 的基本架構 (opens new window)

如果要使用 felx 的排版方式的話，首先我們需要先將元件變成 Flex 屬性，在 CSS 上寫 display: flex，將元件變成 flex container，而裡面的內容就是 flex item，才可以開始我們的排版。

這邊附上課程提供的排版練習，可以依照各個屬性去操作格子 Flexbox Playground (opens new window) 接下來解說也用這個為範例。

# flex-direction (opens new window)

前面提到 Flex 適合用於一維排版，而 flex items 則會有兩種屬性，為列(row) 和 欄（column）主要表示排列的方向

flex-direction: row; 
flex-direction: column;

reverse 顧名思義就是把它相反。

flex-direction: row-reverse;
flex-direction: column-reverse;

# flex-wrap (opens new window)

如果視窗隨著螢幕縮放的話，flex items 很有可能會跑出 container 的情況

所以我們要用 flex-wrap 讓他依照外框去做排列（自動切換到下一排）

flex-wrap: nowrap;
flex-wrap: wrap;
flex-wrap: wrap-reverse;

（其他的設定就如同字面上意思）

# flex-flow (opens new window)

將以上兩種屬性合併使用

.flex-container {
    flex-flow: row wrap;
}

# justify-content (opens new window) 分配 flex items

將 items 水平對齊設定

  • flex-start (default) 水平起始點（置左）

  • flex-end 水平結束的位置（置右）

  • center 水平置中的位置（置中）

  • space-between 最兩側的 items 會貼緊 container 再分配剩餘空間

  • space-around 平均分配所有空間，但並不緊貼 container

  • space-evenly 平均分配空間且沒有剩餘空間
