Linux&Cmd - 常用的幾個命令

Chen Xuan
2021-07-07
  • Linux

紀錄一下常用到的幾個指令（這樣我就不用一直搜尋==）

# 刪除 localhost port

開啟本地測試的話執行兩個相同網頁， port 會衝突，刪除 port 再重新執行即可

kill $(lsof -t -i : port)         //port 放 localhost 的數字

How do I kill the process currently using a port on localhost in Windows? (opens new window)

# Mysql 上的操作

進入 MySql 連線

ssh mysql@10.10.10.10

# 建立檔案、單純看檔案

進入機器上後建立檔案

touch fileName.html

看檔案的話是將 touch 改成 cat

# VM 上的寫入跟儲存

有時候我們會直接在線上機器修改檔案或是看檔案，此時會下

vi fileName

進入後按 i 即是　insert 可以對檔案做修改，修改好後按 ESC 確認無誤後儲存離開的語法為 :wq、:wq! 驚嘆號為強制離開，如果在線上修改太多次他會認為你修改太頻繁（安全性問題）而需要加驚嘆號，單純離開就是 :q、:q

# 刪除機器上的檔案

刪除的話沒有資源回收桶這種東西，請確認好檔案在刪除

rm -r fileName

# 複製檔案至跳板機

先記住要複製的檔案路徑，再到跳板機上輸入以下，記得最後要加上一個點，那代表你的位置

scp mysql@10.10.10.10:/var/www/html/fileName.html .

# 將挑板機檔案丟至機器上

與上面動作相反

scp fileName.html mysql@10.10.10.10:/var/www/html
