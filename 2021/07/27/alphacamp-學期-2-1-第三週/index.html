# 節點 DOM（Document Object Model）

因為 HTML 的結構就像樹狀一樣，而每個分支出去的小樹枝稱為一個節點 DOM，而網頁上有很多 DOM 和 CSSOM 的節點組合成 HTML 而 JavaScript 就是操作節點去改變網頁。其中 Document 是指瀏覽器的 HTML，而 Object Model 就是網頁標籤上的物件

# 樹狀結構：DOM Tree

# 選取節點 DOM

選取節點的方式有很多，像是 querySelector、querySelectorAll、getElementById、getElementByClassName等等用法，自己是最常用 querySelector 去選取 class 名稱或是 tag，也算是普遍主流的用法，而在網頁上尋找節點必須是透過關係，在樹狀結構裡有橫向的「兄弟關係（sibling）」和縱向的「父子關係（parent/child）」

# 陣列迭代 forEach

除了 for 迴圈以外 forEach 是更簡潔的使用，他會將陣列中每一個進行處理，所以使用的話必須放入你要的函式做處理

const age = ['12','23','22','7']
age.forEache(function addYear(){
    age += 1
})
console.log(age) //此時陣列中每一個年紀都被加上 1

而除了函式當參數以外也有其他種

使用上要注意的有三點 一、forEach 沒有回傳值 二、function 的參數可以隨意命名 三、可以使用匿名函式

MDN - Array.prototype.forEach() (opens new window)

# 箭頭函式（arrow function）

只有匿名函式可以使用箭頭函式，範例來說

const triangleArea = function (width, height) {
  return (width * height) / 2
}

改寫為箭頭函式

const triangleArea = (width, height) => { return (width * height) / 2 }

當參數只有一個時，可以省略刮號

const regularTriangle = length => { return (length * length) / 2 }

回傳值只有一行程式碼，可以更省略大括號和 return

const regularTriangle = length => (length * length) / 2

沒有參數時，一定要有括號

const  sayHi = () => 'This is a triangle !'

# 冒泡 Bubbling & 捕獲 Capturing

  • 由下而上的，稱為事件冒泡 (event bubbling)
  • 由上而下的，稱為事件捕獲 (event capturing)

# 事件冒泡 Event Bubbling

事件冒泡指的是從啟動事件的節點 (event.target) 開始，逐層往上傳遞，直到整個網頁的根節點 document 為止。

# 事件捕獲 Event Capturing

當你點擊某個元素時，瀏覽器會先從最高層開始 capturing，一路前往到事件 target，再向上 bubbling

capturing > target > bubbling
