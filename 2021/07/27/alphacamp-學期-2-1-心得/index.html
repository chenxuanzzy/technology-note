Alpha Camp 學習回顧 - 學期2-1：挫折滿滿，收穫滿滿

Chen Xuan
2021-07-27
  • Alpha Camp 筆記
  • 學習回顧

不知不覺來到了第二學期，還記得當初低估 2-1 課程，心裡想著 「哼哼，看我提前把它修完」，結果最後還是乖乖地按照進度一週一週來，課程提前預習不夠，開課後又讀過了一遍，才意識到自己基礎是有多麽不穩QQ。

# 挫敗感的開始

從第一週的 JavaScript 的觀念建立、函式、物件、陣列...等等的使用，到作業「踢掉黑名單」 (opens new window)、「週年慶摸彩」 (opens new window)、「RPG 戰鬥」 (opens new window)，都花了不少時間，邏輯上的建構跟語法的使用在這章節上算是蠻重要的一部分，在練習的過程中感受到許多挫敗感，可能是完成的作業不符合提議、看不懂作業內容、不知道要用什麼語法、邏輯錯誤...等等讓我不禁懷疑自己到底適不適合寫程式，第一週的作業時常花了我一個下午或一整天才完成，不斷的修修改改，觀摩完同學的作業再重新寫一份又會卡住，重頭看了好幾次才比較熟悉，慶幸的是好險自己有花時間提前預讀，在沒有放棄作業的情況下默默的把它完成了（真的花超多時間在作業上）

第二週開始使用套件和網頁最夯的 RWD 應用，其中較難直觀學習的是 Flex 和 Grid，雖然我自己比較喜歡網頁切版的部分，但是屢屢碰到排版的問題也是相當苦惱（挫敗感又隨之襲來），自己花了很多時間在瞭解排版的應用以及觀摩同學作業，查找了一些其他例子和 flex 上的練習。

# 不斷練習 again! 再 again!

第三週開始接觸網頁節點，操控網頁上的 DOM 來製作一個動態網頁，起初開始真的很不熟悉（萬事起頭難！），雖然懂是如何操作（抓到節點）應用（塞回網頁上）的就是不知道語法和有哪些函式可以應用，作業上有幾個都是寫了一遍又練習一次，不斷的加強熟悉度！

有鑒於第三週的加強下，第四週延續第三週的作業，做了彩子的計分版：按鈕功能 (opens new window) 和 My Favorite Movies：按讚與刪除 (opens new window)，做到後面有一種水到渠成的感覺，第四週的作業越做越上手，也知道哪些時候該用哪些語法抓到節點等等。

# 心得

除了明顯感到挫敗以外，也花了很多時間在作業上，過程常常在幫自己加油打氣，自己給自己的鼓勵也很重要，就像我每次看到作業被助教們說是 Excellent 的時候真的是成就感滿滿<3。對於課程上的練習以及一週期間助教的評分批改對於我的個人學習上有明顯的幫助，因為知道只有期限的關係不斷的提醒自己不要拖延，這條路還很長，不斷的督促自己且進步，是我最一開始報名課程的初衷之一，畢竟，不用很厲害才開始，要先開始才會很厲害！
