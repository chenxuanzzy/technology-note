JavaScript 筆記 - Math 的幾種方法

Chen Xuan
2021-07-21
  • JavaScript
  • Math

Math 在計算上還蠻常用到的，像是四捨五入、產生亂數等等..， Math MDN (opens new window) 上的解釋跟方法使用都很詳細，這邊紀錄一下我有用到的幾個方法，可以搭配簡單範例 (opens new window)服用

# Math.floor() 回傳整數

要注意是負數會回傳最近的負整數

Math.floor(3.2) //return 3
Math.floor(5.1) //return 5
Math.floor(-4.2) //return -5

# Math.random() 產生亂數

回傳產生 0 ~ 1 之間的亂數

Math.random() //return 0.83472934

如果想要產生十位數的整數的亂數的話會結合 Math.floor()

let randomInteger = Math.floor(Math.random() * 10) + 1
//如果不加上 1 的話只會產生 0 ~ 9 之間的數字

# Math.abs() 回傳正整數

Math.abs(-5.2) //return 5.2

# Math.round() 四捨五入

這個在實作上蠻常用到的

Math.round(1842.93) //return 1843

# Math.ceil() 取大於或等於的整數

Math.ceil(.95) //return 1
Math.ceil(4) //return 4
Math.ceil(7.004) //return 8
Math.ceil(-7.004) //return 7
