# 目標

數字呈現方式為三位一撇

const answer = [] //儲存轉換後的資料（此時陣列是 String）
const digitNum = [] //儲存轉換後的資料（此時陣列是 Number）
for(let i = 0; i < 5; i++) {
  //產生六位數的亂數
  let randoNumber = Math.floor(Math.random() * 1000000) 
  //轉換成字串再推進 threeDigitString 陣列裡
  digitNum.push((randoNumber).toString())
  //將每一個陣列中的字串用點切割
  let threeDigit = digitNum[i].split('.')
  // console.log(threeDigit)
  //正則表達做去替換三位一個逗點
  threeDigit[0] = threeDigit[0].replace(/\B(?=(\d{3})+(?!\d))/g, ',')
  //推入轉換後的陣列中
  answer.push(threeDigit.join('.'))
}

// console.log(digitNum)
console.log(answer)

想到的做法是放入兩個陣列中，之後再回傳轉換後的陣列，應該還有更優化的方法，這可以算是寫成一個函式庫的感覺拿來應用。
