# Bootstrap 前端框架

源起 Twitter 的工程師，利用 Html、CSS、JavaScript 寫成，核心目標是達成響應式 (opens new window)與行動優先（Mobile First）

# Breakpoint & Mobile First

手機普及化的關係，現在設計網頁除了要依照佈的螢幕尺寸去設計，而手機版本的網頁也是很重要的一點。

  • 行動優先 Mobile First

網頁符合手機、平板、等等行動裝置的話，我們需要以 Mobile First 的原則去撰寫，網站內容如何逐漸變多，而不是單純把同一套內容塞進不同尺寸裡

  • 斷點 Breakpoint

因為要符合行動優先的規則，假設當螢幕畫面縮小到 576px 時，我們的網頁會呈現 576px 設定的樣子，而拉寬至 1200px 也會程先該有的寬度樣子，需要另外多設定一份的做法我們就叫做斷點。

# 媒體類型 Media types

有以下三樣最常使用的為裝置（all）、螢幕（Screen）、印表機（print）

用 Html 表達方式為<link>，但此作法會在 Html 載入時發送多個 HTTP 請求，比較不推薦

<link href = "css/style.css" rel = "stylesheet" type="text/css" media="all">
<link href = "css/style.css" rel = "stylesheet" type="text/css" media="screen">
<link href = "css/style.css" rel = "stylesheet" type="text/css" media="print">

直接寫在 CSS 內，如果@media沒有特別說是哪個裝置的話，預設會是 all

@media (max-width: 600px) {/* 預設裝置為 all */}

@media print {/* 印表機樣式 */}

@media screen and (max-width: 1200px) {/* 螢幕樣式 */}

@media screen and (min-width: 1440px) {/* styles */}

# 媒體查詢 Media Queries

就如同上面斷點所說到的，不同的螢幕要呈現不同的網頁大小樣式，所以開發出了幾項 media feature

max- 為設定寬度高度以下

min- 為設定寬度高度以上

  • min-width : 寬度大於或等於設定值
  • min-height : 寬度小於或等於設定值
  • max-width : 寬度小於或等於設定值
  • max-height : 寬度小於或等於設定值
  • orientation = portrait : 任何高度大於或等於寬度
  • orientation = landscape : 任何寬度大於等於高度

在行動優先的原則下，如果遇到全部都會讀取到的斷點時，因為 CSS 是由上到下讀取所以如果有重複的斷點，放置最下面的則會覆蓋掉其他尺寸

@media screen and (min-width: 400px) {}
@media screen and (min-width: 600px) {}
@media screen and (min-width: 1000px) {}
@media screen and (min-width: 1400px) {} /* 1400px 的會覆蓋掉上面所有*/

# CSS @import

如果有多支 CSS 的檔案的話，我們可以用 import (opens new window) 的方式引入

@import url{'styles.css'} screen and { max-width: 670px };

# 像素 （Pixels）

  • 實體像速與 PPls

實體像素，或稱硬體像素，是指螢幕上會發光的最小硬體單位。 PPls 是每寸像素數（pixels per inch），也就是像素密度，密度越高，肉眼看到更清晰

  • 螢幕解析度

螢幕大小會決定要填滿多少像素，如果螢幕越大像素越少，大螢幕的解析度仍然會很差。

  • CSS 像素

我們在 CSS 裡用到的 px 並不是上述那些像素，它只是一個專門用在網頁發開，是一個無視裝置差別而且等長的測量單位。

# Viewport

Viewport 為網頁內容顯示的區域。 設定 viewport 是要告訴瀏覽器 viewport 的寬度以什麼為基準，以及寬度和 CSS 之間的比例關係。因為 HTML5 普及化的關係，我們要寫網頁時會按驚嘆號跑出預設的基本標籤，而下面這行就是會顯示在預設上，不用自己在額外加上

<meta name="viewport" content="width=device-width, inital-scale=1">
  • width=device-width 調整內容寬度為裝置寬度；
  • initial-scale=1.0 設定 CSS 像素和裝置像素的比例為 1 : 1
