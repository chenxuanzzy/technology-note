# 陣列 Array

  • 有順序性的
  • 可以用 index 找出任一個資料，index 從 0 開始算起

取用陣列內的值方法如以下幾種

const myFriends = ['Alice', 'Lily', 'Cindy', 'Alex']

console.log(myFriends)[0] //此時取到 index 0 所以會回傳 Alice
console.log(myFriends.length - 1) //取到長度後減一則會取到最後一個值 Alex
console.log(myFriends[0] = 'Caleb') //將特定位置改變，此時 index 0 變成 Caleb

# 陣列的幾個常見 method

  • pop() 將最後一個值踢出
  • push() 將值放入陣列中
  • shift() 將第一個值踢出
  • unshift() 將值放入第一個位置

也可以將以上動作存在另一個變數之中，例如以下兩個範例

const nums = [0, 1, 2, 3, 4]

let firstNum = nums.pop() 
console.log(firstNum) // 4

let secondNum = nums.push(8)
console.log(secondNum) // 8

# 物件 Object

宣告一個變數為物件，這種作法叫做 Objcet literal (物件實字)，當然也可以不用宣告直接寫兩個大括號，但通常我們不會這樣做。

const person = {}

# 儲存多種型態的物件

一個物件裡可以存在多種型態的值

const person = {
    //age 為 key, 23 為 value
    age: 23,
    number: ['042426','09836','022231'] //Array
    parent: {
        mother: '50',
        father: '55',
    }
    brush: function(){...}
}

# 查詢與賦予新值

# 查詢方法

person.age //23

person['age']

# 插入新的屬性

person.habit = 'workout'

person['habit'] = 'workout'

# 迴圈列舉物件

有個方法叫做 for-in (opens new window) 它會在物件內的跑一遍

此時的 i 變數會依序儲存物件 person 內的 key，但是這邊要注意一點是如果寫 person['i'] 電腦會跑去找 person 裡 key 為 i 的 key ，所以當想要使用變數當成 key 值的話就要用中括號的方式

for(let i in person) {
    console.log(person[i])
}

# 取得物件資料

  • Object.keys() 取出物件內所有 key
  • Objects.values() 取出物件內所有 value
  • Objects.entries() 取出物件內所有 key 和 vlaue

# 函式 Function()

JavaScript 特殊函式特性

  • 函式可以當成值傳遞（可以放進變數或物件裡）
  • 函式可以當成函式傳遞
  • callback 在特定事件中觸發函式（非同步特性）

# 在物件中呼叫函式

這邊直接使用課程中的例子

const user = {
    name: ['Peter', 'Lee'],
    gender: 'Male',
    age: 41,
    greet: function() {
        console.log(`Hello! I am ${name[0]}.`)
    },
    run: function() {
        console.log('run')
    }
}

consoel.log(user.greet()) //Heelo! I am Peter.
console.log(user.run()) //run!

# 將物件傳給函式

參數傳入如果多的話就會看起來很混亂，所以我們可以直接將整個物件傳入函式的方法

function showDetail(user) {
    console.log(`${user.name[0]} ${user.name[1]}`)
    console.log(user.gender)
    console.log(user.age)
}
showDetail(user)

而當我們新增屬性時，希望他能自動更新物件內容，卻不想印出函式的話可以這樣寫

function showDetail(user) {
    for(let prop in user) {
        if(typeof user[prop] !== 'function') {
            console.log(user[prop])
        }
    }
}
showDetail(user)

# 將物件放入變數

第一種，我們普遍用的方式

function greet(name) {
    return `HI, ${name}`
}

第二種，將一個匿名函式放入變數中

const greet = function() {
    return `HI, ${name}`
}

兩者執行下來是沒有差異的，但是因為 function 有 Hoisting (opens new window) 的特性，且程式是由上往下執行，所以就要特別注意匿名函式所放置或執行的位置

# 將物件當成另一個回傳值

function game () {
  console.log("Let's play game!")
  return "I am happy!"
}
function breakfast () {
  console.log(`I finished my breakfast.`)
  return game()  //把呼叫的函式回傳值當作自己的回傳值
}
const today = breakfast()
console.log(today)

# 將物件當成另一個回傳參數

function game () {
  console.log("Let's play game!")
  return 'I am happy!'
}
//增加這個函式
function study(){
  console.log("Let's go to school.")
  return 'I am tired.'
}
//修改這個函式
function breakfast (callback) {
  console.log(`I finished my breakfast.`)
  return callback() 
}
//呼叫時才決定吃完早餐要去讀書
const today = breakfast(study)
console.log(today)
